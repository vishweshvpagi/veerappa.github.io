Research

Current Projects

Project 1: Acoustic Signal based Fault Detection

This project focuses on using acoustic signals for detecting faults in mechanical systems such as motorcycles. The aim is to develop a reliable and efficient method for early fault detection to prevent major breakdowns and ensure safety.

Project 2: Pattern Recognition in Formal Languages

This research project explores the use of pattern recognition techniques in formal languages to improve the accuracy and efficiency of language processing systems. Applications include natural language processing and computational linguistics.

Project 3: Sound Signal Processing for Noise Reduction

This project aims to develop advanced sound signal processing techniques to reduce noise in various environments. The focus is on creating algorithms that can effectively distinguish between noise and important signals, enhancing audio clarity and quality.
